# Phaser Roguelike Battle Game

Phaser.js 기반의 실시간 전투 시스템과 로그라이크 강화 시스템이 결합된 웹 게임 프로젝트입니다.

## 주요 기능
- **실시간 전투**: 플레이어를 조종하여 적을 피하고, 가장 가까운 적을 자동으로 공격합니다.
- **로그라이크 강화**: 매 층(Floor)을 클리어할 때마다 3가지 무작위 강화 옵션 중 하나를 선택할 수 있습니다.
- **성장 시스템**: 공격력, 체력, 공격 속도, 이동 속도 등 능력치를 영구적으로 강화하여 더 높은 층에 도전하세요.

## 시작하기
1. 의존성 설치:
   ```bash
   npm install
   ```
2. 개발 서버 실행:
   ```bash
   npm run dev
   ```
3. 브라우저에서 `http://localhost:5173` 접속

## 조작 방법
- **이동**: 방향키 (Arrow Keys)
- **공격**: 자동 조준 및 발사

## 프로젝트 구조
- `index.html`: 메인 진입점 및 컨테이너 설정
- `src/main.js`: 게임 로직, 씬(Scene), 상태 관리 핵심 코드
- `package.json`: Phaser 및 Vite 설정
